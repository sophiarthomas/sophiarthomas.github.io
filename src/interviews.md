## Responsive Web Design

Ensures websites automatically adapt to any screen size.

A typical approach would be to implement mobile layout and layer on changes that make sense for sm md lg screens.

- Fluid images
- Fluid grids
- Media tags @media

### Relative length units

- `rem` (Root EM) unit of measurement relative to the root (`<html>`) elements font size
- `em` Relative to the font size of this element
- `vh` `vw` Relative to the viewport's height and width
